# 📱 Form Register Diky (Android Studio Project)

A modern Android registration form application built using **Kotlin** and **Material 3 UI**.  
This project implements complete form validation, user interaction, and modern UI design.

---

## 👨‍💻 Author
**Diky Raihan S**

---

## 🎯 Project Description
This application is a **Complete Registration Form** that includes:
- User input fields (Name, Email, Password)
- Selection controls (Gender, Hobby)
- Spinner (Major selection)
- Validation system
- Alert dialog confirmation
- Gesture interaction (Long Press)

---

## ✨ Features

### ✅ 1. Complete Form
- Full Name
- Email
- Password
- Confirm Password
- Material Design input (TextInputLayout)

---

### ✅ 2. Advanced Validation
- Required field validation
- Email format validation
- Password match validation
- Real-time validation (TextWatcher)

---

### ✅ 3. Selection Controls
- RadioGroup (Gender)
- Checkbox (Hobby - minimum 3 required)

---

### ✅ 4. Spinner & Dialog
- Spinner for Major selection:
  - Industrial Engineering
  - Informatics Engineering
  - Digital Business
  - Retail Management
  - Visual Communication Design
- AlertDialog confirmation before submit

---

### ✅ 5. Gesture Interaction
- Long Press on button for additional action

---

## 🎨 UI Design
- Material 3 Design
- Modern Card Layout

- Clean and responsive UI
- Aesthetic blue theme
